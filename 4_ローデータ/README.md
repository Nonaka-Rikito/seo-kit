# 4. ローデータ (Raw Data)

全カテゴリに共通する基礎的なデータ。業界調査、競合分析、市場統計、会議記録など。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `competitor-analysis/` | 競合他社の分析データ |
| `industry-research/` | 業界動向・AIニュースサマリーなど |
| `market-statistics/` | 市場統計データ |
| `mtg-transcripts/` | 会議の文字起こし・議事録 |

## 活用方法
これらのデータは特定のクライアントに紐づかない、汎用的なリサーチ素材です。
各プロジェクトで参照する際の共通ソースとして活用してください。
