# 5. 機密情報 (Confidential)

AIツールに自動で読み取らせたくない機密データを格納するフォルダ。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `service-account-keys/` | GCPサービスアカウントキー（JSON） |
| `finance/` | 請求書、見積書、契約書、経費データ（PDF/CSV/Numbers） |

## 重要
- このフォルダ内のファイルは `.gitignore` で全てGit管理対象外です
- AIツールでの自動参照を避けるため、明示的に指示しない限り読み込まれません
- 機密性の高いファイルは必ずこのフォルダに格納してください
