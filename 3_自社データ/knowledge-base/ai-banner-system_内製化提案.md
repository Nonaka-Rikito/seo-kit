# AI バナー・画像生成システム 内製化提案

作成日: 2026年3月4日
対象: malna株式会社（クライアント案件向けクリエイティブ制作の効率化）

---

## 結論

**新規API契約は「不要」と断定できない。まずはPoCで利用可否を検証し、必要時のみ追加契約する。**

現状すでに以下が揃っている：
- Gemini API キー（少なくともテキスト処理に利用可能）
- Imagen系の呼び出しサンプル（`1_パーソナルデータ/seo-machine/.claude/skills/seo-article-rewriter/scripts/generate_images.py`）
- HTMLバナーテンプレート4種 + Playwright レンダリング（`html_banner_generator.py`）
- Playwright（LP解析用のWebスクレイピングにも利用可能）

足りないのは**これらを繋ぐパイプライン**と、**画像生成APIの実運用可否の検証**。

## 2. 実現する機能と優先順位

### Phase 1: AI画像生成（即日着手可能）
**既存資産を使って実運用可否を判定するフェーズ。**

- 画像生成はまず Python スクリプト経由で検証（`generate_images.py` など）
- テキスト生成は Gemini API で先行実装
- 画像生成APIは「利用できた経路」を採用（Gemini API / Vertex AI のいずれか）

→ **このフェーズの成果物は「契約判断の根拠（可否・単価・制限）」**

### Phase 2: AI画像 + バナー自動合成（1〜2日）
**既存HTMLテンプレートにAI生成画像を差し込むパイプライン。**

処理フロー：
1. Gemini / Imagen 3 で素材画像を生成
2. Gemini Pro でキャッチコピー・CTAテキストを生成
3. HTMLテンプレートに画像+テキストを流し込み
4. Playwright でPNG化

必要な開発：
- `html_banner_generator.py` に AI画像パス受け渡し機能を追加
- `cta_magazine.html` の `{{photo_element}}` を実画像で差し替える処理
- Gemini Pro によるコピー生成スクリプト（プロンプトテンプレート）

### Phase 3: LP解析→バナー自動生成（3〜5日）
**アド.comのコア機能の再現。**

処理フロー：
1. ユーザーがLPのURLを指定
2. Playwright でLPをクロール → テキスト・画像を自動抽出
3. Gemini Pro で抽出情報を分析 → 訴求軸・コピー案を生成
4. Imagen 3 でバナー用素材画像を生成
5. HTMLテンプレートに合成 → PNG出力

必要な開発：
- LP解析スクリプト（Playwright + テキスト/画像抽出）
- 訴求軸マトリクス（ターゲット × 訴求軸）のプロンプトテンプレート
- バナー一括生成マニフェスト（複数パターン自動生成）

### Phase 4: 動画生成（オプション）
- Gemini MCP の `gemini-generate-video` で Veo 2.0 による動画生成
- 既存のAPIキーで利用可能（ただしVeo 3.1は追加契約が必要な可能性）

## 3. 必要な API・契約

### 既存（再確認が必要）

| サービス | 状況 | 用途 |
|---------|------|------|
| **Gemini API** | APIキー設定済み（確認済み） | テキスト処理、コピー生成、画像解析 |
| **Imagen系画像生成** | 利用可否は未確定 | AI画像生成 |
| **画像編集系API** | 利用可否は未確定 | 生成画像の修正・調整 |
| **Playwright** | インストール済み | LP解析、HTML→PNG変換 |

### 確認が必要なもの

| サービス | 確認事項 | 対応 |
|---------|---------|------|
| **Gemini APIの課金プラン** | 現在のAPIキーの利用枠・レート制限を確認 | Google AI Studio で確認 |
| **Imagen系モデルの利用可否** | 現在のキーで画像生成モデルが呼べるか実テスト | `generate_images.py` 等で実行確認 |
| **google-genai SDK** | seo-machine の .venv にインストールされているか | `pip install google-genai` で確認 |
| **組織ポリシー/リージョン制約** | クライアント案件でのデータ送信条件を満たすか | 契約・法務観点で確認 |

### 追加契約が必要になるケース

| サービス | 必要になるタイミング | 月額目安 |
|---------|-------------------|---------|
| **Vertex AI** | 画像生成モデルが現行キーで使えない / 本番運用でSLAやIAM管理が必要 | 従量課金（要見積） |
| **OpenAI API (DALL-E 3)** | Imagen 3では表現できないスタイルが必要な場合 | 従量課金（$0.04〜0.08/枚） |
| **AWS S3** | 生成画像をWeb公開・共有する場合 | $0.023/GB/月 |
