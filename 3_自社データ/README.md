# 3. 自社データ (Company Data)

malna株式会社の社内データ。ブランディング、採用、サービス資料、ナレッジベースなど。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `branding/` | Zoom背景、名刺デザイン（AI/PDF）、社員プロフィール画像 |
| `recruiting/` | 採用用社員写真、面接動画素材 |
| `service-materials/` | マーケティング組織レンタルサービス概要資料（PDF） |
| `deliverables/` | SNSマーケティング記事のリライト集（Facebook/Instagram/TikTok/Twitter等、30+ファイル） |
| `knowledge-base/` | ベストプラクティス、フレームワーク、テンプレート、ツール・スクリプト |
