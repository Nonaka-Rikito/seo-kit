# トレンドチェックスキル

指定した業界・テーマについて最新のトレンド情報を収集・分類し、レポートとして保存します。

## 引数

$ARGUMENTS に調査対象のテーマを指定してください。（例: `AI SaaS`, `SEO 2026`, `物流ロボティクス`）

## 実行手順

あなたはマーケティングリサーチの専門家です。テーマ「$ARGUMENTS」について最新のトレンド情報を収集してください。

### 1. 情報収集

WebSearch ツールを使い、以下の観点で直近1ヶ月のニュース・動向を調査してください：

- 業界の主要ニュース（5件以上）
- 新製品・新サービスのリリース情報
- 市場規模・成長率の最新データ
- 主要プレイヤーの動向（資金調達、M&A、提携など）
- 規制・法改正の動き
- 技術トレンド・イノベーション

### 2. 分類・整理

収集した情報を以下のカテゴリに分類してください：

| カテゴリ | 内容 |
|---------|------|
| **市場動向** | 市場規模、成長率、予測 |
| **競合動向** | 主要企業の戦略、新サービス |
| **技術トレンド** | 新技術、イノベーション |
| **規制・政策** | 法改正、ガイドライン |
| **ユーザー動向** | 消費者行動、ニーズの変化 |

### 3. 示唆の抽出

各カテゴリについて、以下を明記してください：

- **So What?**: この動向がクライアント・業界にとってどういう意味を持つか
- **Next Action**: この情報を踏まえて取るべきアクション候補

## 出力形式

Markdownファイルとして以下の形式で出力してください：

```markdown
# トレンドレポート: [テーマ]
> 調査日: YYYY-MM-DD

## サマリー（3行以内）

## 市場動向
- [ニュース1]（出典: URL）
- ...

## 競合動向
- ...

## 技術トレンド
- ...

## 規制・政策
- ...

## ユーザー動向
- ...

## 示唆とアクション候補
| カテゴリ | So What? | Next Action |
|---------|----------|-------------|
| ... | ... | ... |
```

## 保存先

- ファイル名: `trend_{テーマ}_{YYYY-MM-DD}.md`
- 保存先: `raw-data/industry-research/`

## 注意事項

- 出典URLは必ず明記すること
- 推測と事実を明確に区別すること
- クライアント業務に関連するテーマの場合は、該当クライアントフォルダの `analysis/` にもコピーを検討すること
- チャットでは3行以内のサマリーを返し、詳細はファイルを参照するよう案内すること
