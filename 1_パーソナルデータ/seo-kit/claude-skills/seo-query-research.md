# SEO クエリ調査スキル

対象キーワードについてSEO観点から徹底的な調査を行い、記事作成の基盤となる情報をまとめます。

## 引数

$ARGUMENTS に対象キーワードを指定してください。（例: `オウンドメディア`）

## 実行手順

あなたはSEOとコンテンツマーケティングの専門家です。キーワード「$ARGUMENTS」について徹底的な調査をお願いします。

WebSearch ツールを活用して、実際の検索結果上位サイトの情報を収集してください。

以下の情報を詳細に分析してください：

1. このキーワードで検索するユーザーの属性（年齢層、性別、職業、関心事など）
2. 検索意図の分類（情報収集型、比較検討型、購入意図型など）
3. ユーザーが抱える具体的な悩みや問題点
4. 上位5サイトが提供している情報と解決策の共通点
5. 上位5サイトがユーザーに提示しているゴール
6. 関連する長尾キーワード上位20個と検索ボリューム予測
7. コンテンツで取り上げるべき重要なトピック

できるだけ具体的かつ詳細な情報を提供してください。推測ではなく、検索エンジンの上位表示サイトから得られる客観的な情報を基に分析してください。

## 出力形式

最終的には以下の形式で調査結果をまとめてください：

```
- ターゲット読者: [調査で判明したユーザー属性を詳細に記述]
- 検索意図: [調査で判明した検索意図を詳細に記述]
- 解決すべき課題: [ユーザーの悩みや問題点を詳細に記述]
- 提供すべきゴール: [上位サイトが提示しているゴールを詳細に記述]
- 関連キーワード: [関連する長尾キーワード上位20個と検索ボリューム予測]
- 重要トピック: [コンテンツで取り上げるべき重要なトピックのリスト]
```

上記の形式に加えて、上位5サイトの分析内容も詳しく記載してください。

## 注意事項

- この調査結果は次のステップ（`/seo-structure` で構成作成）に引き継がれます。会話の中でそのまま続けて使用してください。
- WebSearch で実際の検索結果を取得し、客観的なデータに基づいて分析してください。
- 推測だけでなく、検索結果から得られた具体的な事実を明記してください。
