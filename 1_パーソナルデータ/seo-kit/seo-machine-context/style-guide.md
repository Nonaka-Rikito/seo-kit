# jo-katsu.com（ジョーカツ）スタイルガイド

このガイドは jo-katsu.com の全コンテンツに適用する文章表記・フォーマット規則を定義する。
brand-voice.md（トーン・ボイス）、seo-guidelines.md（SEO基準）と併用し、一貫性を確保すること。

---

## 文体・敬語

### 基本文体
- **です・ます調**を基本とする（常体「だ・である」は原則禁止）
- ただし、箇条書き内の体言止めは可（例: 「面接マナーの基本」）
- 見出し（H1-H4）は体言止めまたは名詞句（例: 「自己PRの書き方」「面接で聞かれる質問5選」）

### 人称
- **一人称**: 使用しない（メディアとしての客観性を保つ）
- **二人称**: 「あなた」は原則使わず、文脈で読者を示す
  - OK: 「面接では第一印象が重要です」
  - NG: 「あなたは面接で第一印象を大切にしましょう」
- **三人称**: 体験談では「Aさん」「Bさん」等のイニシャル表記

### 文の長さ・段落
- **1文**: 40〜80文字を目安。100文字を超える場合は分割する
- **1段落**: 3〜4文で構成
- **能動態優先**: 受動態は必要な場合のみ使用

## 漢字・ひらがな・カタカナの使い分け

### 文字種バランス（目標値）
- **漢字比率**: 20-35%
- **ひらがな比率**: 35-55%
- **カタカナ**: 外来語・固有名詞・強調に限定

### ひらがなで書く語（漢字を開く）

以下の語は、漢字ではなくひらがなで表記する:

| 漢字表記（使わない） | ひらがな表記（使う） |
|---|---|
| 事 | こと |
| 物 | もの |
| 時 | とき |
| 所 | ところ |
| 為 | ため |
| 出来る | できる |
| 致します | いたします |
| 下さい | ください |
| 頂く | いただく |
| 宜しく | よろしく |
| 様々 | さまざま |
| 更に | さらに |
| 但し | ただし |
| 即ち | すなわち |
| 及び | および |
| 尚 | なお |
| 又は | または |
| 予め | あらかじめ |
| 殆ど | ほとんど |
| 暫く | しばらく |

### 漢字で書く語（閉じる）

以下の語は漢字で表記する:

| ひらがな表記（使わない） | 漢字表記（使う） |
|---|---|
| おこなう | 行う |
| あらわす | 表す |
| かんがえる | 考える |
| つたえる | 伝える |

### カタカナ表記ルール
- 外来語はカタカナ: エントリーシート、インターンシップ、ガクチカ
- 長音記号「ー」を使う: コンピューター（○）、コンピュータ（×）
- ただし、業界慣用がある場合はそちらに従う: サーバー（○）、サーバ（×）

## 就活用語の表記統一

### 使う表記 → 使わない表記

| 使う | 使わない | 備考 |
|---|---|---|
| エントリーシート | ES | 初出時は「エントリーシート（ES）」と併記。2回目以降は「ES」可 |
| 面接官 | 面接する人 | |
| 内定 | 内々定 | 正式には異なるが、一般読者向けには「内定」で統一 |
| 自己PR | 自己ピーアール | |
| ガクチカ | 学生時代に力を入れたこと | 初出時はフル表記で説明を添える |
| 就活 | 就職活動 | 初出時は「就職活動（就活）」と併記可 |
| OB・OG訪問 | OB訪問 | 性別を限定しない表記にする |
| 志望動機 | 志望理由 | 表記を統一する |
| グループディスカッション（GD） | グルディス | 初出はフル表記。略称はGDのみ可 |
| Web面接 | ウェブ面接 / WEB面接 | 「Web」の表記で統一 |

### 略語ルール
- **初出**: フル表記（略語）の形式で記載する
  - 例: 「エントリーシート（ES）」「グループディスカッション（GD）」
- **2回目以降**: 略語のみ使用可
- **広く認知された略語**: 以下は初出からそのまま使用可
  - ES、GD、OB・OG、PR、SPI、IT

## 数字・日付・時間の表記

### 数字の書き方
- **算用数字を使用する場面**:
  - 具体的な数値: 3社、5つのポイント、10分
  - パーセンテージ: 30%（「パーセント」と書かない）
  - 金額: 5,000円、1万円
  - 年月日: 2025年6月1日
  - 順序・ランキング: 第1位、1番目
- **漢数字を使用する場面**:
  - 慣用句・固有表現: 一石二鳥、一般的、第一印象、一人ひとり
  - 概数: 数十社、数百人

### 大きな数字
- 4桁以上はカンマ区切り: 1,000、10,000
- 万以上は漢字併用: 1万人、100万円（10000人、1000000円とは書かない）
- 「約」は数字の前に: 約3,000社

### 日付の書き方
- **年月日**: 2025年6月1日（西暦を使用。和暦は原則使わない）
- **月日のみ**: 6月1日
- **年度**: 2025年度、25卒（「25年卒」でも可）
- **期間**: 2025年3月〜6月（全角チルダ「〜」を使用。半角「~」は不可）

### 時間の書き方
- **時刻**: 14:00、9:30（24時間表記）
- **所要時間**: 約30分、1時間30分
- **期限表現**: 「6月1日まで」「6月1日締切」

## 見出し・リスト・表のフォーマット

### 見出し（Heading）
- **H1**: 1記事に1つ。メインキーワードを含める
- **H2**: 4-6個。記事の大きな区分。読者がH2だけ読んでも記事全体の流れがわかること
- **H3**: H2の下に2-4個。詳細項目
- **H4**: H3の下に配置。必要な場合のみ
- **階層スキップ禁止**: H2→H4のようにH3を飛ばさない
- **見出しの文体**: 体言止めまたは疑問形
  - OK: 「面接マナーの基本」「なぜ自己分析が必要なのか」
  - NG: 「面接マナーの基本について説明します」（冗長）

### 箇条書き（リスト）
- **使用基準**: 3項目以上の列挙は箇条書きにする
- **文頭の統一**: 項目の書き出しを揃える（並列構造）
  - OK: 「企業研究をする」「自己分析をする」「ESを書く」
  - NG: 「企業研究」「自己分析をしましょう」「ESの執筆」
- **句読点**: 文が完結する場合は句点をつける。体言止めの場合は不要
- **番号付きリスト**: 手順や優先順位がある場合に使用
- **ネスト**: 最大2階層まで

### 表（テーブル）
- **使用基準**: 比較や対照が明確な場合に使用
- **ヘッダー行**: 必ず設ける
- **セル内**: 簡潔に。長文は避ける
- **例**: 企業比較、選考スケジュール、対策項目一覧

## リンクの書き方

### アンカーテキスト
- **具体的な表現を使う**:
  - OK: 「面接マナーの詳細はこちらの記事で解説しています」→ 「面接マナーの基本」にリンク
  - NG: 「詳しくはこちら」「ここをクリック」
- **キーワードを含める**: SEO上有効なアンカーテキストにする
- **文字数**: 5〜15文字程度

### リンク配置ルール
- **内部リンク**: 3-5本（seo-guidelines.md準拠）
- **外部リンク**: 1-3本（公的機関、大手就活サイトの公式ページ等）
- **外部リンク**: 新しいタブで開く設定にする

## 句読点・記号

### 句読点
- **句点**: 「。」を使用（ピリオド「.」は使わない）
- **読点**: 「、」を使用（コンマ「,」は使わない）
- **文末**: です・ます調の文は必ず「。」で終える

### 括弧
- **丸括弧**（ ）: 補足説明、略語の併記
  - 例: エントリーシート（ES）
- **鉤括弧**「 」: 引用、固有名詞の強調、用語の初出
  - 例: 「ガクチカ」とは学生時代に力を入れたことを指します
- **二重鉤括弧**『 』: 書籍名、作品名
  - 例: 『就活の教科書』

### その他の記号
- **中黒**「・」: 並列（例: OB・OG訪問）
- **全角チルダ**「〜」: 範囲（例: 3月〜6月）
- **感嘆符・疑問符**: 見出しでは使用可。本文では控えめに使用する
  - 見出し例: 「面接で聞かれる質問5選」「なぜ自己分析が必要？」
  - 本文: 疑問符の後はスペースなしで続ける（日本語の慣例に従う）

## 太字・強調の使い方

### 太字（Bold）
- **使用場面**: 重要なキーワード、結論、数値の強調
- **頻度**: 1セクション（H2区間）あたり2〜4箇所が目安
- **乱用禁止**: すべてを太字にすると効果がなくなる
- 例: 「面接では**第一印象が最も重要**です。最初の**3〜5秒**で印象が決まると言われています。」

### イタリック
- 日本語コンテンツでは原則使用しない（日本語フォントでは効果が薄い）
- 英語の書籍名・作品名に限り使用可

### 下線
- リンクと混同するため使用しない

### 全角大文字
- 日本語に該当なし。英語の略語（SEO, ES, GD等）はそのまま使用

## 禁止表現・注意表現

### 禁止表現
以下の表現は使用しない:

| 禁止表現 | 理由 | 代替表現 |
|---|---|---|
| 絶対に受かる | 根拠なき断定 | 合格率を高める方法 |
| 必ず内定が取れる | 根拠なき断定 | 内定獲得の可能性を高める |
| 〇〇するだけで | 過度な簡略化 | 〇〇を中心に取り組む |
| 裏ワザ・裏技 | 信頼性を損なう | 効果的なテクニック |
| ぶっちゃけ | 品位を損なう | 実際のところ |
| ヤバい | 品位を損なう | 注意が必要 / 重要 |
| こちらをクリック | SEO・アクセシビリティ上NG | 具体的なアンカーテキスト |
| 筆者は〜 / 私は〜 | 一人称禁止 | （主語を省く、または客観的表現に） |
| あなたは〜 | 二人称の直接使用 | 文脈で読者を示す |

### 注意表現（使用時に配慮が必要）
- **「〜と言われています」**: 出典を明記する。出典なしの場合は避ける
- **「〜が一般的です」**: データや根拠を示す
- **「最近では〜」**: 具体的な時期や年度を示す（例: 「2025年卒の就活では〜」）
- **「多くの企業が〜」**: 具体的な数値や調査結果を示す（例: 「約70%の企業が〜」）

### 差別・偏見を含む表現の禁止
- 性別を限定する表現を避ける（「女子就活」→「就活生向け」）
- 特定の学歴を前提とした表現を避ける
- 容姿に関する具体的指示を避ける（「髪型は黒髪で」→「清潔感のある身だしなみ」）

## コンテンツ構成テンプレート

### 導入部（リード文）
- **文字数**: 200〜300文字
- **構成**:
  1. **共感・問題提起**（1〜2文）: 読者の悩みや疑問に触れる
  2. **記事の概要**（2〜3文）: この記事で何がわかるかを示す
  3. **行動の約束**（1〜2文）: 読み終えたら何ができるようになるかを示す
- **キーワード配置**: メインキーワードを冒頭200文字以内に含める

### 本文セクション
- **1セクション（H2単位）**: 300〜500文字
- **500文字を超える場合**: H3で小見出しを入れて分割する
- **小見出し間隔**: 300〜400文字ごとに配置

### まとめ・結論
- **文字数**: 200〜400文字
- **構成**:
  1. **要約**（箇条書き3〜5項目 or 段落）: 記事の要点を振り返る
  2. **次のアクション**（1〜2文）: 読者が今すぐすべきことを示す
  3. **CTA**（1〜2文）: 関連記事への誘導やサービス紹介
- **メインキーワードをまとめにも含める**

## 画像・メディア

### alt属性（代替テキスト）
- 画像の内容を簡潔に説明する
- 可能であればキーワードを自然に含める
- 「画像」「写真」で始めない（スクリーンリーダーが自動で伝えるため）
- 文字数: 40文字以内を目安

### キャプション
- 必要に応じて画像の下に配置
- 体言止めで簡潔に

### スクリーンショット
- 操作手順の説明には積極的に使用する
- 関連部分のみをトリミングする
- 必要に応じて矢印や囲みで注目箇所を示す

## データ・統計の引用ルール

### 引用の書き方
- **出典を必ず明記**: 「〇〇の調査によると」「〇〇（出典名、年）」
- **年度を明記**: データの時点を示す（例: 「マイナビ2025年卒調査」）
- **リンクを付ける**: 可能な限り一次ソースへリンクする

### 数値の表記
- 大きな数字は丸める: 「約1万2,000人」（12,345人とは書かない）
- パーセント記号を使用: 30%（「30パーセント」とは書かない）
- 金額は円記号: 5,000円（「五千円」とは書かない）
- カンマ区切り: 10,000（10000とは書かない）

---

## アクセシビリティ

- **見出し階層を守る**: H1→H2→H3の順。スキップしない
- **リンクテキスト**: 「こちら」ではなく具体的な表現を使う
- **画像のalt属性**: 全画像に設定する
- **色だけに依存しない**: 色分けだけで情報を伝えない
- **読みやすさ**: 漢字比率20-35%、1文40-80文字を守る
