# 1. パーソナルデータ (Personal Data)

野中さん自身に関するデータ。思考・価値観・個人的な学習メモ・自作ツールなど。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `思考/` | 思考メモ、Slack関連のリサーチノート |
| `有料記事/` | 有料コンテンツの下書き・素材 |
| `Google-API連携/` | Google Analytics / GSC / Drive / Calendar のAPI連携スクリプト |

## 注意事項
- サービスアカウントキー（JSONファイル）は `5_機密情報/` に格納済み
- node_modules は `npm install` で再生成可能
