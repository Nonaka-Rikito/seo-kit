# example-media.com SEOガイドライン

このドキュメントはexample-media.comの全コンテンツに適用するSEO基準を定義する。

## 日本語SEO基準

### タイトルタグ
- **文字数**: 28-32文字（Google日本語表示の上限目安）
- **構成**: メインキーワード + サブ情報 + ブランド名
- **例**: 「面接マナーの基本｜知らないと損する5つのポイント」
- **NG例**: 「就活のいろいろなコツについてまとめました」（曖昧、キーワード不明確）

### メタディスクリプション
- **文字数**: 100-120文字（モバイル表示を考慮）
- **構成**: 結論 → 具体的内容 → 行動喚起
- **例**: 「面接マナーの基本を5つのポイントで解説。入室から退室まで、面接官に好印象を与える振る舞いがわかります。例文付きで今すぐ実践可能。」

### URLスラッグ
- ローマ字またはキーワードの英訳
- ハイフン区切り、短く簡潔に
- 例: `/campus/mensetsu-manner-kihon`

## 見出し構成

### H1
- 1つのみ。タイトルタグと同義または拡張版
- メインキーワードを含める

### H2
- **4-6個**で記事の大きな区分を構成
- 1/3以上にメインキーワードまたは関連語を含める
- 読者がH2だけ読んでも記事全体の流れがわかるようにする

### H3
- H2の下に**2-4個**。詳細項目
- H2→H4のスキップは禁止（必ずH3を経由する）

## キーワード最適化

### キーワード密度（形態素ベース）
- **メインキーワード**: 1.0-2.0%
- **サブキーワード**: 0.3-0.8%
- **詰め込み上限**: 3.0%を超えたら警告

### 重要配置ポイント
メインキーワードを必ず以下に含める:
- [ ] H1見出し
- [ ] 冒頭200文字以内
- [ ] H2見出しの1/3以上
- [ ] まとめ・結論セクション
- [ ] タイトルタグ
- [ ] メタディスクリプション

### キーワード統合のベストプラクティス
- 自然な日本語として読めることを最優先
- 同義語・類義語のバリエーションを使う
  - 「就職活動」→「就活」→「就活準備」
- 質問形式を含める
  - 「面接で何を聞かれる？」

## コンテンツ品質基準

### 文字数要件
- **標準記事**: 3,000-6,000文字（目標: 4,000-5,000文字）
- **まとめ記事**: 5,000-8,000文字
- **ニュース・速報**: 1,500-2,500文字
- 文字数を稼ぐための水増しは厳禁

### 内部リンク
- **最低**: 3本
- **推奨**: 4-5本
- **上限**: 7本（3,000文字以上は例外）
- アンカーテキストは具体的に（「こちら」は禁止）

### 外部リンク
- **推奨**: 1-3本
- 公的機関、大手就活サイトの公式ページ、信頼性の高いソース
- 統計データの引用時は必ずソースリンクを付ける

## 日本語特有の注意事項

### 文字種バランス
- **漢字比率**: 20-35%（高すぎると読みにくい）
- **ひらがな比率**: 35-55%
- **カタカナ**: 外来語や強調に限定

### 可読性
- **1文**: 40-80文字（100文字超は分割）
- **1段落**: 3-4文
- **箇条書き**: 3項目以上の列挙は箇条書きに変換
- **小見出し**: 300-400文字ごとに配置

### E-E-A-T（経験・専門性・権威性・信頼性）
- **経験**: 実際の就活体験に基づく情報、事例
- **専門性**: 採用市場のデータ、HR専門家の知見
- **権威性**: 公的機関のデータ、大手調査結果の引用
- **信頼性**: 最新データの使用、透明性のある情報提供

## 就活メディア固有の指標

### 季節変動パターン
- **1-3月**: インターンシップ・自己分析系
- **3-6月**: ESエントリーシート関連が急上昇
- **6-9月**: 面接対策系が最盛期
- **10-12月**: 内定辞退・就活やり直し系
