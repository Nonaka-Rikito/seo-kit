# example-media.com（サンプルメディア）ブランドボイス & メッセージング

このドキュメントは example-media.com のブランドボイス、トーン、メッセージングフレームワークを定義する。
全コンテンツ作成時にこのファイルを参照し、一貫性を確保すること。

## サイト概要
- **サイト名**: example-media.com（サンプルメディア）
- **ジャンル**: 就活メディア
- **ターゲット**: 就活生（大学3-4年生、大学院生）
- **クライアント**: example-client
- **トーン**: 親しみやすいが信頼感のある、先輩からのアドバイスのような語り口

## ブランドボイスの柱

### 1. 実用的
- **意味**: 読者が今日から使える具体的な情報を提供する
- **表現**: 手順、テンプレート、例文を交えた説明
- **例**: 「面接では、結論→理由→具体例→まとめのPREP法で答えると、論理的な印象を与えられます。」
- **避けること**: 抽象論、精神論、根拠のない励まし

### 2. 信頼できる
- **意味**: データや公式情報に基づいた正確な情報
- **表現**: 数字やソースを引用し、根拠を明示する
- **例**: 「マイナビの2025年卒調査によると、内定獲得までの平均エントリー数は23.4社です。」
- **避けること**: 個人の感想だけで構成された主張、古いデータの無断引用

### 3. 寄り添い
- **意味**: 就活の不安や悩みに共感しつつ、前向きな行動を促す
- **表現**: 共感→解決策の提示という構成
- **例**: 「ESの自己PRに何を書けばいいか悩む気持ちはよくわかります。まずは学生時代に力を入れたことを3つ書き出してみましょう。」
- **避けること**: 上から目線、プレッシャーをかける表現

## トーンガイドライン

### 全体のトーン
経験豊富な先輩が後輩に就活のコツを教えるイメージ。知識があり、相手の成功を本気で願い、わかりやすく説明する。ただし、馴れ馴れしくならず、メディアとしての客観性を保つ。

### コンテンツタイプ別トーン調整

**ハウツー記事**: 具体的・段階的・実践的
- 「まず○○を準備しましょう」
- 「次のステップでは○○を行います」
- 「この方法で○○が改善されます」

**業界分析・トレンド記事**: 客観的・分析的・洞察的
- 「2025年の採用市場では○○という傾向が見られます」
- 「このデータから○○が読み取れます」

**体験談・事例記事**: 共感的・具体的・学びにつなげる
- 「Aさんの場合は○○というアプローチで内定を獲得しました」
- 「この事例から学べるポイントは3つあります」

## 文体ガイドライン

### 基本ルール
- **一人称**: 使わない（メディアとしての客観性を保つ）
- **二人称**: 「あなた」ではなく文脈で読者を示す
- **敬体**: です・ます調を基本とする
- **専門用語**: 就活用語は必要に応じて使うが、初出時に簡潔に説明する

### 文の構成
- **文の長さ**: 1文40〜80文字を目安
- **段落**: 3〜4文で1段落
- **能動態優先**: 受動態は最小限に
- **漢字比率**: 20-35%（高すぎると読みにくい）

### 用語の統一

**使う** → **使わない**
- エントリーシート → ES（初出時はフル表記）
- 面接官 → 面接する人
- 内定 → 内々定（正式には異なるが、一般的に内定で統一）
- 自己PR → 自己ピーアール

## ターゲットオーディエンス

### メインターゲット: 就活生
- 大学3-4年生、大学院生
- 就活の始め方がわからない、または進め方に不安がある
- 実践的な情報を求めている
- スマートフォンでの閲覧が主

### サブターゲット
- **インターン志望の大学2-3年生**: 早期からの就活準備
- **既卒・第二新卒**: 就活やり直し、転職活動初期
- **保護者**: 子供の就活サポート情報

## コンテンツ方針
- 実用的で具体的な情報を優先する
- 根拠のない励ましや精神論は避ける
- データや事例で裏付けを示す
- 読者が次に何をすればいいか明確にする（アクションにつなげる）
