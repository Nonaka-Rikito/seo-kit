## 3. ノベルティ詳細プラン

### A. ばらまき用：花王バブ入浴剤

| 項目 | 詳細 |
|------|------|
| 単価 | 〜90円 |
| 数量 | 1,000個 |
| 小計 | 約90,000円 |
| 仕様 | バブ1錠 + オリジナルチラシA封入（OPP袋） |
| 配布方法 | コンパニオンが通路で声がけ配布 |

**選定理由**:
- 「お疲れさまです」の一言で自然に渡せる
- 情シスの身体的疲労に寄り添うメッセージ性
- 受け取り心理のハードルが極めて低い（消耗品・個包装）
- チラシ同封でサービス認知を獲得

### B. 良いノベルティ：ウェブカメラカバー（名入れ）

| 項目 | 詳細 |
|------|------|
| 単価 | @258円 |
| 数量 | 200個 |
| 小計 | 約51,600円 |
| 仕様 | スライド式カバー / josysロゴ＋URL名入れ / チラシB同封 |
| 配布方法 | 名刺交換後のお礼として手渡し |

**選定理由**:
- **セキュリティ担当者に最も刺さるアイテム**（プライバシー保護の実用品）
- PCに常時貼付 → **毎日josysを想起**する接触頻度
- josysの「セキュリティを守る」メッセージと完全に合致
- 社内会議・Web会議で他の社員の目にも触れる（社内拡散効果）
- 「josysはセキュリティの会社」というブランド連想を形成

### B'. 次点候補：めぐりズム 蒸気でホットアイマスク

| 項目 | 詳細 |
|------|------|
| 単価 | @162円 |
| 数量 | 200個 |
| 小計 | 約32,400円 |
| 仕様 | めぐりズム1枚 + チラシB封入（OPP袋） |

**評価**: 体験価値は高いが、一度使うと消えるため継続想起に弱い。カメラカバーとの併用は予算次第で検討。

### その他候補（参考）

| アイテム | 単価 | 評価 |
|---------|------|------|
| 名入れドリップコーヒー | 〜150円 | 消耗品。話題にはなるが想起に弱い |
| 電子メモパッド | 〜500円 | 面白いが単価が高く、情シス文脈との接続が薄い |
| ガジェットクリーナー | 〜200円 | 実用的だがインパクトに欠ける |
| プリングルズ（名入れ） | 〜300円 | SNS映えするが展示会文脈では浮く |

## 7. 営業トークスクリプト

### Scene 1: 通路での声がけ（コンパニオン）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 シーン: 通路を歩く来場者に対して
 配布物: バブ入浴剤 + チラシA
 目標:  受け取ってもらう（認知獲得）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

コンパニオン:
  「お疲れさまです！
   情シスのみなさんに入浴剤をお配りしています。
   よかったらどうぞ！」

  ※ 受け取ったら ※
  「ありがとうございます。SaaS管理でお困りのことが
   あれば、あちらのブースでご相談いただけます！」

  ※ 断られそうなら ※
  「入浴剤だけでもぜひ！チラシも入ってますので、
   お時間あるときにご覧ください」
```

**ポイント**:
- 「情シスのみなさんに」と**ターゲット限定のフック**を入れることで、該当者の足を止める
- 「営業トーク」ではなく「ねぎらい」で入るため、受け取りのハードルが低い
- 無理な引き止めはしない。チラシで後追いできる設計

### Scene 3: ヒアリング〜デモ誘導（営業）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 シーン: 会話が始まった来場者との商談
 目標:  課題特定 → デモ体験 → 名刺交換
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【パターンA: SaaS管理に課題がある場合】

営業:
  「なるほど、Excelで台帳管理されているんですね。
   実は先月リリースした機能で、連携するだけで
   社内のSaaSを自動検出できるようになったんです。

   30秒だけお時間いただければ、実際の画面を
   お見せできますが、ご覧になりますか？」

  ※ デモ後 ※
  「ちなみに、シャドーITのリスクスコアも
   自動でA〜Eの5段階評価されるので、
   経営層への報告資料もすぐ作れます。

   よろしければお名刺を交換させていただいて、
   後日、御社の環境に合わせた詳しいご案内を
   お送りしてもよろしいですか？」

【パターンB: 特に課題を感じていない場合】

営業:
  「管理されているんですね、素晴らしいです。
   ちなみに、退職者のアカウント削除って、
   人事異動のタイミングで漏れなくできてますか？」

  ※ 「まあ、たまに...」と反応があれば ※
  「そこなんです。josysだと人事システムと連携して、
   退職と同時にアカウントを自動停止できるんです。
   よかったら画面だけでもご覧になりますか？」
```

**ポイント**:
- 「30秒だけ」と時間のハードルを下げる
- デモは「全機能紹介」ではなく「1つの課題解決」にフォーカス
- 名刺交換は「後日のご案内」という理由づけで自然に

### Scene 4: 名刺交換後のクロージング（営業）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 シーン: 名刺交換完了後
 配布物: ウェブカメラカバー + チラシB
 目標:  記憶に残る体験 + 後追い商談の布石
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

営業:
  「ありがとうございます。
   お礼にこちらをお持ちください。」

  （カメラカバーを渡す）

  「ウェブカメラのセキュリティカバーです。
   josysのロゴが入ってますが、
   PCに貼ってお使いいただけます。

   SaaSのセキュリティを守るのが私たちの仕事ですが、
   まずはあなたのPCのプライバシーから
   お守りさせてください。」

  （チラシBを示しながら）

  「裏面に情シスの課題チェックリストがあるので、
   社内で当てはまる項目がないか
   確認していただけると嬉しいです。

   2つ以上当てはまったら、ぜひデモ申込の
   QRコードからお申し込みください。
   展示会にお越しいただいた方には
   優先枠をご用意しています。」
```

**ポイント**:
- カメラカバーを渡す際の「セキュリティを守る」トークがjosysのサービスメッセージと直結
- チラシBのチェックリストを「社内で確認してもらう」ことで、**展示会後の社内共有を促進**
- 「展示会限定の優先枠」で行動を促す
