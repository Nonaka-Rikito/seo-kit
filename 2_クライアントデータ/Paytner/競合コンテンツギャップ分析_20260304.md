## 7. 補足：データソース制限について

- **Ahrefs API**: Lite 2022プランのためAPI経由でのデータ取得不可。検索ボリュームはaramakijake.jp（Google/Yahoo推定値）を使用
- **現在順位データ**: 既存分析（2/24）のAhrefs Web UIデータを参照。最新の順位データ取得にはAhrefs Web UIでの再確認を推奨
- **「推定」表記のあるボリューム**: 類似キーワードからの推計値。正確な数値はGoogle Keyword PlannerまたはAhrefs Web UIで確認推奨
- **KD（キーワード難易度）**: Ahrefs APIが使えないため「高/中/低」の定性評価。既存分析ではKD 0-5が多く、全体的に低難易度の領域
