# _adhoc/ — 一時的な分析スクリプト・データファイル

ルートに散在していたアドホックなファイルを集約。
必要に応じて適切なプロジェクトフォルダに移動すること。

## 内容

### 分析スクリプト
- `add_bom.py` - CSV BOM付与
- `analyze_viplimo.py` - VIP Limo分析
- `convert_csv.py` - CSV変換
- `parse_ahrefs.py` - Ahrefs出力パース
- `run_analysis.py` - 分析実行
- `run_parse.bat` - パース実行バッチ
- `run_topic_analysis.bat` - トピック分析バッチ

### データファイル
- `gsc_rewrite_candidates_campus.csv` - GSCリライト候補
- `gsc_rewrite_top50_new_candidates.csv` - GSC Top50候補
- `malna_gsc_data.csv` - malna GSCデータ
- `ahrefs_analysis_output.txt` - Ahrefs分析出力

### その他
- `creative-banner1.html` / `creative-banner2.html` - クリエイティブバナー
- `GA4-GSC-measurement-system-complete-summary.md` - GA4/GSC計測システムまとめ
- `選定画像30枚/` - 画像選定ファイル
- `画像選定レポート.xlsx` - 画像選定レポート
