# Bannerbear テンプレート設定依頼

## 背景
Bannerbear（bannerbear.com）でCTAバナーのテンプレートを作成中です。テンプレートエディタが開いている状態です。現在、以下のレイヤーは設定済みです。

**設定済み:**
- `bg` — 矩形 / 1920x1080 / 色#F0EEEB / 位置(0,0)
- `accent_bar` — 矩形 / 1920x12 / 色#E8384F / 位置(0,0)
- `headline_prefix` — Text「どこよりもリアルな」/ Noto Sans JP / 36px / 色#333333 / 位置(60,30)
- `headline` — Text「企業分析資料」/ Noto Sans JP Bold / 96px / 色#1A1A1A / 位置(60,120)

## 依頼内容
以下の残りレイヤーを、テンプレートエディタで1つずつ設定してください。画面を見ながら、次に何をクリック・入力するかを1ステップずつ具体的に指示してください。

## 残りのレイヤー設計書

### 1. `badge`（右上の社会的証明バッジ）
- 種類: Text
- テキスト: 「10,000人の学生の本音を凝縮」
- フォント: Noto Sans JP Bold
- サイズ: 32px
- 文字色: #FFFFFF（白）
- 背景色（Background）: #1A1A1A（黒）※テキストの背景色機能をONにする
- 背景の角丸（Background Radius）: 8
- 位置: x=1200, y=30
- レイヤー名: `badge`

### 2. `cta_bar`（下部のCTA帯の背景）
- 種類: Rectangle（矩形）
- サイズ: 幅1920 x 高さ140
- 位置: x=0, y=940
- 色: #E8E4E0
- レイヤー名: `cta_bar`
- レイヤー順序: `bg`の上、テキストレイヤーの下

### 3. `cta_badge`（「無料」バッジ）
- 種類: Text
- テキスト: 「無料」
- フォント: Noto Sans JP Bold
- サイズ: 28px
- 文字色: #E8384F（ピンク赤）
- 背景色: なし（もし枠線を付けられるなら #E8384F で1pxの枠線）
- 位置: x=60, y=975
- レイヤー名: `cta_badge`

### 4. `cta_text`（CTAメインテキスト）
- 種類: Text
- テキスト: 「資料をダウンロードする >>」
- フォント: Noto Sans JP Bold
- サイズ: 42px
- 文字色: #1A1A1A
- 位置: x=200, y=970
- レイヤー名: `cta_text`

### 5. `logos_image`（企業ロゴ群の画像エリア）
- 種類: Image
- 位置: x=60, y=380
- サイズ: 幅680 x 高さ320
- デフォルト画像: 今は空でOK（後でAPIから画像URLを差し替える）
- レイヤー名: `logos_image`

### 6. `document_image`（右側のレポート表紙画像）
- 種類: Image
- 位置: x=950, y=180
- サイズ: 幅550 x 高さ600
- デフォルト画像: 今は空でOK（後でAPIから画像URLを差し替える）
- レイヤー名: `document_image`

## 注意事項
- 各レイヤーは追加後に必ずレイヤー名をリネームしてください（APIから参照するキーになります）
- レイヤーの重なり順序は下から: bg → cta_bar → accent_bar → その他テキスト・画像
- フォントが「Noto Sans JP」で見つからない場合は「Noto Sans CJK JP」等で検索してください
- 全レイヤーの設定が終わったら「Save」してください

## 完了確認
全レイヤーを設定し終わったら、テンプレートのプレビューを見せてください。参考クリエイティブ（就活生向け企業分析資料のダウンロードCTAバナー）に近い配置になっていればOKです。
